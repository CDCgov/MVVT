# MVVT: Immune Repertoire Analysis

## Overview

This repository contains analysis workflows, scripts, and Jupyter notebooks developed by the Molecular Virology and Vaccines Team (MVVT) for the analysis of adaptive immune receptor repertoire sequencing (AIRR-seq) data. The workflows support analyses of B-cell receptor (BCR) and antibody repertoire sequencing data generated from bulk and single-cell experiments.

Individual analysis projects are organized into separate subdirectories, each containing project-specific workflows, notebooks, documentation, and example input files where appropriate.
